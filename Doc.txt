Best Strategy

Do NOT only watch tutorials.

For every topic:

Learn concept
Build small feature
Break it
Fix errors
Rebuild without watching

That is how backend actually sticks.

What to focus on for your new fullstack job

Since you already know React:

Learn Express fast
Learn API creation
